# Linux Cheat Sheet

A simple website with a list of useful commands for Linux.

## Get Started

1. Clone the repository
```bash
git clone https://github.com/ItsLeon15/Linux-Cheat-Sheet.git
```

2. Run the website
```bash
cd Linux-Cheat-Sheet
npx http-server
```
